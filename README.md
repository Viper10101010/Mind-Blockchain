# Mind Blockchain Project

![Mind Blockchain Logo](assets/images/logo.png)

Mind Blockchain is a **decentralized blockchain platform** focused on **mental health innovation, secure data management, and global collaboration**. The platform leverages blockchain technology to create a secure, transparent, and privacy-focused environment for mental health professionals, researchers, and patients.

This repository contains a **reconstructed version of the Mind Blockchain website** as it appeared around **May 22, 2023**, along with assets and documentation.
